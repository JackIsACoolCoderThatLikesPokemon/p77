Visual Studio Code is a IDE made by the tech giant Microsoft for coding in all sorts of formats (including HTML, JS, Python, C++, Java, json and more.)



The console is an interface used by developers to test code and see outputs. You can access the console by using Inspect Element and can type any JS code into it. You can print stuff on the console by using 'console.log("text")' or 'console.log(var)'
